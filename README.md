A Python CLI dice game with authentication and SQLite storage.

## Features
- User registration/login
- Secure password hashing (Argon2)
- SQLite persistence
- CLI interface using argparse

## Tech stack
- python 3.13
- argon2
- sqlite3
- argparse

## Usage
# CLI
dicegame signup
dicegame login
dicegame delete user
dicegame view users
dicegame view scores
dicegame display
dicegame guess
